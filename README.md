## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14.x or later)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hija-happy/WedWise.git
   cd WedWise
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   ```bash
   http://localhost:5173
   ```
